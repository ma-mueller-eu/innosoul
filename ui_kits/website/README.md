# InnoSoul Website UI Kit

## Overview
High-fidelity clickable prototype of the InnoSoul Consulting marketing website.

**Design width:** 1280px (responsive down to mobile)  
**Language:** German (primary)  
**Stack:** React + Babel (inline JSX)

## Screens / Sections
1. **Hero** — Headline, subline, dual CTA, trust signals
2. **Leistungen** — 5 service cards with icons
3. **Branchen** — MedTech, Biotech, Life Science focus areas
4. **Über uns** — Team profiles (Magdalena + Nicole)
5. **Referenzen** — 1 real + 2 placeholder success stories
6. **Kontakt** — Contact form + Calendly CTA

## Components
- `Header.jsx` — Sticky nav with logo, links, CTA button
- `Hero.jsx` — Dark navy hero with headline split, dual CTA, trust badges
- `Services.jsx` — Services grid with icon cards
- `Industries.jsx` — Branche focus section
- `Team.jsx` — Team cards with bios
- `References.jsx` — Success story cards
- `Contact.jsx` — Contact form section
- `Footer.jsx` — Footer with links

## Notes
- Profile photos not available — placeholder portraits used
- Logo asset: `../../assets/logo.webp`
